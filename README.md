# The-Movie-Watchlist-Project
## Languages & Technologies Used

- HTML
- CSS
- JavaScript
- REST APIs (OMDb API / TMDB API / Watchmode API)

## Core Features

- Live Search (real-time movie search using API)
- Watchlist Toggle (add/remove movies from saved list)
- Empty States UI (no results / empty watchlist handling)
- Responsive Dark Theme UI (Disney+ / HBO Max style)
- Sticky Sidebar (to display saved/watchlist movies)
